## Install Instructions

### Quick Check
1. Is at least Java 1.8 installed? <br/>
2. Does executing <b> java -version </b> on command line / command prompt return 1.8 or greater? <br/>
3. If not, please install Java 8+ and make it the default. (Java 7 will also work at this time) <br/>

## Installation for ApkTool
### Windows:
1. Download Windows <a href="https://raw.githubusercontent.com/iBotPeaches/Apktool/master/scripts/windows/apktool.bat">wrapper script</a> (Right click, Save Link As <b>apktool.bat</b>) <br/>
2. Download apktool-2 (<a href="https://bitbucket.org/iBotPeaches/apktool/downloads/">find newest here</a>) <br/>
3. Rename downloaded jar to <b>apktool.jar</b> <br/>
4. Move both files (<b>apktool.jar</b> & <b>apktool.bat</b>) to your Windows directory (Usually <b>C://Windows</b>) <br/>
5. If you do not have access to <b>C://Windows</b>, you may place the two files anywhere then add that directory to your Environment Variables System PATH variable. <br/>
6. Try running <b>apktool</b> via command prompt <br/>
### Linux:
1. Download Linux <a href="https://raw.githubusercontent.com/iBotPeaches/Apktool/master/scripts/linux/apktool">wrapper script</a> (Right click, Save Link As <b>apktool</b>) <br/>
2. Download apktool-2 (<a href="https://bitbucket.org/iBotPeaches/apktool/downloads/">find newest here</a>) <br/>
3. Rename downloaded jar to <b>apktool.jar</b> <br/>
4. Move both files (<b>apktool.jar</b> &  <b>apktool</b>) to /usr/local/bin (root needed) <br/>
5. Make sure both files are executable (chmod +x) <br/>
6. Try running <b>apktool</b> via cli <br/>
### macOS:
1. Download Mac wrapper script (Right click, Save Link As apktool) <br/>
2. Download apktool-2 (find newest here) <br/>
3. Rename downloaded jar to apktool.jar <br/>
4. Move both files (apktool.jar & apktool) to /usr/local/bin (root needed) <br/>
5. Make sure both files are executable (chmod +x) <br/>
6. Try running apktool via cli <br/>
